Miscellaneous Metas
================================
These are several miscellaneous metagames that don't require a seperate mod folder, they are contained in the ```formats.js``` file.
Introduction
--------------------------------

List ofmetagames contained:
* [Level Balance][1]

Installing
--------------------------------
You can copy a block of metagame code to your ```formats.js``` file in your Pokémon Showdown server's ```config/formats.js``` (for instance, ```~/Downloads/Pokemon-Showdown/formats.js``` or ```C:\Users\Bob\Downloads\Pokemon-Showdown\config\formats.js```)
